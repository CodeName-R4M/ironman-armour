# ironman-armour
A Flask web application for a secret login system inspired by Tony Stark's Iron Man armor.

## Overview
This project is a simple Flask web application that simulates a secret login system similar to the authentication mechanisms found in Tony Stark's Iron Man armor. Users can log in with a specific username and password to unlock a hidden feature.

## Features
- Simulated secret login system
- Cookie-based authentication
- Dashboard access upon successful login

## Tech Stack / Built With
- HTML
- Python
- Flask

## Installation & Setup
```bash
git clone https://github.com/CodeName-R4M/ironman-armour.git
cd ironman-armour
pip install -r requirements.txt
python app.py
```

## Usage
```python
# To run the Flask application
python app.py
```

## Project Structure
- `app.py`: Main Flask application file
- `templates/`: Directory containing HTML templates
